Iserción de Imágenes

Un elemento muy importante en una página web o documento html son las imágenes o gráficos, ya que como dice el refrán "una imagen dice más que mil palabras". Utilizaremos la etiqueta < img src = url >.

Los formatos de imágen más utilizados son:

  • .jpg
  • .png
  • .gif

Fondo de Página

El fondo de página se refiere a personalizar con un color o imágen lo que estara detrás del contenido, se utilizan las etiquetas "BG COLOR" para emplear un color sólido, y la etiqueta "BACKGROUND=URL" para usar una imágen. Cabe mencionar que esta etiqueta se debe aplicar dentro de < BODY > para reflejarlo en el navegador. La única desventaja que tiene es que no podemos personalizar los tamaños en la utilización del "BACKGROUND". Para esto es necesario trabajar desde las hojas de estilo "CSS". Aplicaremos un color sólido para ejemplificar y después una imágen.
